# Agents

## Checking your work
As you write code, you can check your work by running `make check`. This will run the following checks:
- Linting
- Security
- Testing (unit tests and e2e tests)
- Code coverage

`bash kira_e2e_tests.sh` will run the e2e tests and are worth running
